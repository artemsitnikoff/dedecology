# Журнал работ — ЭкоПульс

Хронология того, что сделано по проекту админки **ЭкоПульс** (сбор обращений о площадках ТКО). Прототип построен по образцу проекта «Глафира».

## Что в проекте

| Файл / папка | Что это |
|---|---|
| `ЭкоПульс.dc.html` | Основной прототип (живой, на дизайн-системе «Аркадий CRM») |
| `ЭкоПульс (standalone).html` | Та же админка одним самодостаточным файлом (офлайн, для показа без сервера) |
| `react-app/` | Классический React-проект для передачи разработчикам (отдельные `.jsx`, `ReactDOM.createRoot`) |
| `docs/ТЗ - ЭкоПульс.md` | Полное техническое задание |
| `docs/Claude Code — выжимка.md` | Краткая выжимка для вставки в Claude Code |

Все три версии (DC, standalone, react-app) держатся синхронно.

## История изменений

1. **Базовая админка** — список инцидентов, сайдбар, статусы (Новый / Инцидент обнаружен / Нет инцидента / Выгружен), источники (Макс / Яндекс форма), поиск, фильтры, выгрузка в Excel (CSV), карточка инцидента, чекбоксы и массовые действия.
2. **Только таблица** — вид «карточки» убран, оставлена плотная таблица; воронка статусов 1-в-1 как в Глафире.
3. **Столбцы** зафиксированы в порядке и переразбиты: Фото · Дата · Время · Регион · Город · Адрес · Координаты · Статус · Источник · Чат. Сортировка — по клику на заголовок (стрелки ▲▼), верхний выпадающий сортировщик убран.
4. **Регион/Город/Адрес** разделены на отдельные поля и столбцы.
5. **Настройки** — раздел: профиль (ФИО + email), сброс пароля (одно поле «Новый пароль» + «Сбросить»), пользователи и доступ, приглашение по email (как у Глафиры).
6. **Период-фильтр** — календарь «с — по» вместо кнопок «неделя/месяц».
7. **React-проект** `react-app/` — разложен на классические `.jsx`-компоненты для разработчиков.
8. **Ребренд → ЭкоПульс** — зелёное сердце-смайлик 💚 вместо деда (логотип, пустое состояние, карточка); тексты, email, имена выгрузок.
9. **Блок «Дата · время»** — фото, дата и время объединены в один **замороженный** блок слева (sticky, тень), сортировка единым ключом дата+время.
10. **Карточка как у Глафиры** — выезжает **справа над областью таблицы** (закрывает Регион/Город/Адрес/Координаты…), не перекрывая воронку и фильтры; окно не затемняется. При открытии — **экран загрузки** с пульсом 💚 и надписью «Загрузка…», затем контент.
11. **ID + телефон** добавлены в левый блок (под датой/временем), в карточку и в выгрузку Excel; пульс оставлен только справа (загрузка + карточка).
12. **Компактная шапка** — вертикальные отступы шапки/поиска/воронки/фильтров уменьшены ~на 30%, таблица поднята выше.
13. **Логика статусов** — блок «Сменить статус» вынесен наверх карточки и стал контекстным (только доступные переходы); одиночное фото не растягивается.
14. **Полировка** — кнопки «Сменить статус» уменьшены и сделаны белыми (как чипы воронки); пульс логотипа ЭкоПульс возвращён.
